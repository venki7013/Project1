What is CSS?
CSS (Cascading Style Sheets) is used to styles web pages. Cascading Style Sheets are fondly referred to as CSS. 
The reason for using this is to simplify the process of making web pages presentable. It allows you to apply styles on web pages. 
More importantly, it enables you to do this independently of the HTML that makes up each web page.
